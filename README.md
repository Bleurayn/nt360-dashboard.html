# NT360 RealTime Clinical Intelligence — Public Prototype (v1.0)

**Live demo**: https://bleurayn.github.io/nt360-dashboard.html  
**Live version**: Open `nt360-dashboard.html` in any browser

### What this public repository contains
- A standalone, client-side HTML/CSS/JS dashboard that demonstrates the NT360 user experience
- Real-time integration with the public ClinicalTrials.gov API (v2, August 2025)
- Responsive design, live trial stream, basic analytics visualization

### What this repository does NOT contain (and never will)
- Any backend code, servers, or infrastructure
- Any machine-learning models or proprietary algorithms
- Any private APIs, customer data, EHR integrations, or paid data sources
- Any authentication system or user data handling
- The commercial NT360 platform (which is fully closed-source and proprietary)

> **The real NT360 product — including the fusion engine, predictive AI models, multi-source data reconciliation, IRB monitoring, and all revenue-generating features — is 100% proprietary, closed-source, and developed in private repositories.**
